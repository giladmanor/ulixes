Ulixes.io @UlixesEngine

The Open Source Adaptive Rule Engine, That Can!

Download this project as a .zip file Download this project as a tar.gz file

A few important things you need to know about the Ulixes rule engine:

  • It is open source and free
  • It is easy to install and start using
  • It is supported by a group of master engineers

What Can it Do? (Out of the box features)

  • Provides the standard PBL (Points, Badges, Levels) facilities as every other gamification solution
  • Implements a Behavior Flow thus enabling high definition personalization
  • Implements a messaging system for immediate communication and responses to the users
